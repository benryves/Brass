.branchrule

.branchrule rule

Sets the rule used to create branch table labels for TI apps. {*} indicates the label itself. Defaults to _{*}

.branchrule _{*}
.branch Label ; creates branch label _Label

.branchrule {*}!
.branch Label ; creates branch label Label!

.branchrule `{*}`
.branch Label ; creates branch label `Label`
